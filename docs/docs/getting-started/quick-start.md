# Quick Start Guide

Get up and running with MINA in minutes.

## Installation

MINA is distributed as a desktop application. After installation, launch the app to get started.

## Your First Workflow

Let's create a simple automation workflow:

1. Navigate to **Automation Circuit**
2. Click the **Workflows** tab
3. Click **New Workflow**
4. Give it a name (e.g., "Daily System Check")
5. Add steps using the workflow editor
6. Save and enable the workflow

## Monitoring Your System

1. Go to **System Monitor**
2. View real-time CPU, memory, disk, and network metrics
3. Check running processes
4. Monitor network connections

## Market Intelligence

1. Visit **Stock News** to see the latest market news
2. Use **Market Data** to track stock prices
3. Create a **Portfolio** to manage your investments
4. Check the **Economic Calendar** for important events

## Getting Help

- Click the `?` button in any module for contextual help
- Visit the **Documentation** section for detailed guides
- Check the API reference for programmatic access

## Tips

- Use the command bar (`Cmd/Ctrl+K`) for quick navigation
- Enable real-time updates for live data
- Create automation workflows to save time
- Customize your dashboard layout

Ready to learn more? Explore the module-specific documentation!
