# Getting Started Overview

Welcome to MINA! This guide will help you get up and running quickly.

## First Steps

1. **Explore the Dashboard**: Start by opening MINA and exploring the main dashboard. You'll see all available modules organized by category.

2. **Foundation Modules**: These provide system monitoring and configuration:
   - System Monitor
   - Network Constellation
   - DevOps Control
   - Security Center

3. **Intelligence Modules**: Access market data and analysis:
   - Reality & Timeline (Temporal Engine)
   - Stock News
   - Market Data
   - Portfolio Manager

4. **Documentation**: Use the Documentation section to learn about each module in detail.

## Navigation Tips

- **Sidebar**: Use the left sidebar to navigate between modules
- **Command Bar**: Press `Cmd+K` (Mac) or `Ctrl+K` (Windows/Linux) to open the command bar for quick navigation
- **Help Buttons**: Many modules have a `?` button in the header for quick help
